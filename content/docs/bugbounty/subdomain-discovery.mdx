# Subdomain Discovery

Discover subdomains using various reconnaissance tools and techniques.

## Basic Subdomain Enumeration

<CommandBlock
  title="Subfinder Basic Scan"
  description="Uses Subfinder to discover subdomains for a target domain."
  command="subfinder -d example.com -silent"
/>

<CommandBlock
  title="Assetfinder Discovery"
  description="Find subdomains using Assetfinder tool."
  command="assetfinder --subs-only example.com"
/>

## Advanced Enumeration

<CommandBlock
  title="Subfinder with Custom Wordlist"
  description="Enhanced subdomain discovery with custom wordlists and resolvers."
  command="subfinder -d example.com -w /usr/share/wordlists/subdomains.txt -r /root/resolvers.txt -silent"
/>

<CommandBlock
  title="Subdomain Brute Force"
  description="Brute force subdomains using a comprehensive wordlist."
  command="gobuster dns -d example.com -w /usr/share/seclists/Discovery/DNS/subdomains-top1million-110000.txt -t 50"
/>

## Live Domain Validation

<CommandBlock
  title="HTTP Probe Validation"
  description="Check which discovered subdomains are live and accessible."
  command="cat subdomains.txt | httprobe -c 50 | tee live_subdomains.txt"
/>

<CommandBlock
  title="Combined Discovery Pipeline"
  description="Complete subdomain discovery and validation pipeline."
  command="subfinder -d example.com -silent | assetfinder --subs-only | sort -u | httprobe -c 50 | tee final_subdomains.txt"
/>
